# flutter_test_app

Main Features:
- Display Product List
    - Searching (Based on product's name)
    - Filtering (Based on categories and ratings)
- Display Product Details
    - Zoom in and out of item's image
    - Add and remove item to cart
 
Notable Packages:
- BLoC (For State Management)
- Secure Storage (To store the data of items added to cart)
- Cached Network Image (To reduce images load time)

Configurations:
- Flutter Version: 3.32.8
- Dart Version: 3.8.1
